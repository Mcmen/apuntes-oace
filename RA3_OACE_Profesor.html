B41

Introducción a LibreOffice Writer · Entorno de trabajo

⏱️ 110 min
🧠 Solución test rápido — B41

Respuesta correcta: ✅ C) .odt

💡 .odt es el formato nativo de Writer (OpenDocument Text). .docx es de Microsoft Word, .txt es texto plano sin formato, .pdf es para distribución.

📋 Solución ejercicio 3.1 · Explora Writer
  • ¿En qué barra se encuentra el botón "Guardar"? → En la barra de herramientas estándar (la superior, junto a Nuevo y Abrir).
  • ¿Cómo se cambia el tipo de letra de un texto seleccionado? → Desde la barra de herramientas de formato, desplegable de fuentes.
  • ¿Qué indica la tecla "INS" en la barra de estado? → Alterna entre modo INSERCIÓN (inserta letras sin borrar) y SOBRESCRITURA (reemplaza caracteres).
  • ¿Qué extensión tiene por defecto un archivo de Writer? → .odt
🎓 Nota didáctica

Muchos alumnos vienen de Microsoft Word y confunden extensiones. Insiste en que .odt es el formato estándar de LibreOffice, aunque puede exportar a .docx y .pdf.

⚠️ Errores frecuentes

No diferenciar entre "Guardar" (Ctrl+G) y "Guardar como". Explica la diferencia desde el primer día.

B42

Writer · Formato de carácter y párrafo

⏱️ 110 min
🧠 Solución test rápido — B42

Respuesta correcta: ✅ B) Ctrl + N

💡 Ctrl + N (Negrita), Ctrl + K (Cursiva), Ctrl + S (Subrayado). Es importante que memoricen estos atajos.

📋 Solución ejercicio 3.2 · Aplica formato
  • Título: "El formato de texto..." → negrita, tamaño 14, centrado.
  • Párrafo completo: justificado (Ctrl+J), interlineado 1.5 (Formato → Párrafo → Interlineado → 1.5 líneas).
  • Palabra "legibilidad": seleccionar → Ctrl+K (cursiva) + Ctrl+S (subrayado).
🎓 Nota didáctica

El formato de párrafo (justificado, interlineado) suele confundirse con el de carácter. Haz ejercicios donde tengan que aplicar AMBOS para que vean la diferencia.

B47

Writer · Repaso integrado y prueba final

⏱️ 110 min
📋 Solución ejercicio 3.7 · Informe completo con Writer

El informe debe contener:

  • Portada: título, autor, fecha (formato centrado, tamaño 20-24).
  • Índice automático: generado a partir de Título 1 y Título 2.
  • Secciones: mínimo 3 con estructura jerárquica.
  • Tabla de ventajas/desventajas: mínimo 3 filas, bordes y formato.
  • Imagen con ajuste "Ajuste": texto alrededor.
  • Encabezado y pie: título y número de página.
  • PDF exportado: Archivo → Exportar como PDF.

Exportación a PDF correcta1 pts

B50

Calc · Fórmulas y funciones básicas (SUMA, PROMEDIO)

⏱️ 110 min
🧠 Solución test rápido — B50

Respuesta correcta: ✅ C) =SUMA(B2:B10)

💡 En LibreOffice Calc, la función SUM en inglés es SUMA en español.

📋 Solución ejercicio 3.10 · Factura sencilla
Subtotal (D2): =B2*C2
Subtotal (D3): =B3*C3
TOTAL (D4): =SUMA(D2:D3)

Para copiar la fórmula de D2 a D3, basta arrastrar el cuadro de relleno (esquina inferior derecha).

Resultado: =SI(E2>=5; "Aprobado"; "Suspenso")

B55

Calc · Repaso integrado (Presupuesto)

⏱️ 110 min
📋 Solución ejercicio 3.15 · Presupuesto mensual completo
TOTAL INGRESOS: =SUMA(B2:B3)  (suponiendo Salario en B2, Extras en B3)
TOTAL GASTOS: =SUMA(B5:B11)   (todos los gastos)
AHORRO: =B1-B4                (Total Ingresos - Total Gastos)
SI: =SI(AHORRO>200; "Bien"; "Revisar")

Gráfico circular: Seleccionar los nombres de gastos y sus valores → Insertar gráfico → Circular.

Valores esperados: Ingresos=2000€, Gastos Fijos=755€, Gastos Variables=510€, Total Gastos=1265€, Ahorro=735€ → "Bien".

Tablas requeridas:

  • Socios: ID_socio (PK), Nombre, Teléfono, Email, Dirección
  • Libros: ID_libro (PK), Título, Autor, Año, ISBN

B63

Impress · Transiciones y animaciones

⏱️ 110 min
🧠 Solución test rápido — B63

Respuesta correcta: ✅ B) Transición es entre diapositivas, animación entre objetos

📋 Solución ejercicio 3.23 · Animaciones
  • Transición: Vista normal → seleccionar diapositiva → panel derecho "Transición de diapositiva" → elegir disolvencia → aplicar a todas.
  • Animación título: Seleccionar título → Animación personalizada → Añadir → Entrada → Volar.
  • Animación imagen: Seleccionar imagen → Añadir → Entrada → Aparecer.
  • Duración: En el panel de animación/transición, ajustar "Duración" a 0.5-1 segundo.
🎓 Nota didáctica

Las animaciones son efectivas si se usan con moderación. Una presentación con demasiadas animaciones distrae al público.

B64

Proyecto final RA3 · Presentación profesional integradora

⏱️ 110 min
📋 Solución · Presentación "Soluciones informáticas OACE"

Contenido mínimo esperado:

  • 10 diapositivas
  • Portada con título, autor y fecha
  • Índice con enlaces (opcional, pero recomendado)
  • Integración de tabla desde Writer/Calc (copiar y pegar manteniendo formato)
  • Gráfico desde Calc (copiar como imagen o como objeto OLE)
  • Datos de clientes desde Base (copiar tabla o vincular)
  • Transiciones y animaciones discretas
  • Exportación a PDF

📊 Rúbrica de corrección de la prueba final RA3 (20 pts)

FaseCriterioPts
Estructura10 diapositivas, portada, índice, cierre4 pts
ContenidoProductos, servicios, equipo, conclusiones4 pts
Integración Writer/CalcTabla (Writer/Calc) + Gráfico (Calc)4 pts
Integración BaseDatos de clientes desde Base3 pts
Transiciones/AnimacionesCoherentes, discretas, funcionales3 pts

🎓 Nota didáctica

Esta prueba evalúa la integración de todas las herramientas ofimáticas. Los alumnos pueden copiar tablas de Writer/Calc a Impress; lo importante es que sepan cómo hacerlo y mantener el formato.
