Introducción a LibreOffice Writer · Entorno de trabajo y primeros pasos

⏱️ 110 min
RA3 CE-a · Describir funciones del procesador de textos (5%)

LibreOffice Writer es el procesador de textos del paquete ofimático LibreOffice. Su función principal es crear, editar y dar formato a documentos de texto como cartas, informes, currículums o trabajos académicos. Es una alternativa gratuita y de código abierto a Microsoft Word.

🎥 Vídeo · Conoce LibreOffice Writer
📌 Elementos del entorno de Writer
  • Barra de menús: Archivo, Editar, Ver, Insertar, Formato, Tabla, Herramientas...
  • Barras de herramientas: Estándar (guardar, abrir, imprimir) y Formato (negrita, alineación, tipo de letra).
  • Regla: Para ajustar márgenes y tabulaciones.
  • Área de trabajo: Donde se escribe el texto.
  • Barra de estado: Muestra número de página, idioma, modo de inserción (INS).

📁 Formatos de archivo

  • .odt → Formato nativo de Writer (OpenDocument Text)
  • .docx → Compatible con Microsoft Word
  • .pdf → Para distribuir sin posibilidad de edición

⌨️ Atajos de teclado básicos

  • Ctrl + G → Guardar
  • Ctrl + A → Abrir
  • Ctrl + P → Imprimir
  • Ctrl + Z → Deshacer
  • Ctrl + Y → Rehacer
🔑 Concepto clave

Writer es un procesador de textos, no un programa de diseño gráfico ni una hoja de cálculo. Está especializado en trabajar con texto, pero permite añadir imágenes, tablas y otros objetos para enriquecer los documentos.

Writer · Formato de carácter y párrafo · Negrita, alineaciones, interlineado

⏱️ 110 min
RA3 CE-c · Formatear texto (10%)RA3 CE-b · Crear, modificar y manipular documentos (15%)
🎥 Vídeo · Formato de texto en Writer

El formato de carácter afecta a palabras sueltas: tipo de letra (fuente), tamaño, negrita, cursiva, subrayado, color. El formato de párrafo afecta a bloques de texto: alineación, interlineado, sangrías y espacios antes/después.

✏️ Formato de carácter (atajos)

  • Ctrl + N → Negrita
  • Ctrl + K → Cursiva
  • Ctrl + S → Subrayado
  • Ctrl + M → Quitar formato directo

📐 Formato de párrafo (atajos)

  • Ctrl + L → Alinear a la izquierda
  • Ctrl + E → Centrar
  • Ctrl + R → Alinear a la derecha
  • Ctrl + J → Justificar
📌 Interlineado

El interlineado es el espacio entre líneas de un mismo párrafo. Se configura en Formato → Párrafo → Sangrías y espacios → Interlineado. Los valores típicos son: Sencillo (1), 1.5 líneas y Doble (2).

Writer · Estilos y plantillas · Mantén la coherencia visual

⏱️ 110 min
RA3 CE-b · Crear, modificar y manipular documentos (15%)
🎥 Vídeo · Estilos en Writer
📌 ¿Qué es un estilo?

Un estilo es un conjunto de formatos (tipo de letra, tamaño, color, interlineado, etc.) que se guarda con un nombre. Al aplicar el estilo a un texto, se aplican todos los formatos a la vez. Si modificas el estilo, todos los textos que lo usan se actualizan automáticamente.

📂 Tipos de estilos en Writer

  • Estilos de párrafo: Afectan a párrafos completos (título, subtítulo, cuerpo de texto).
  • Estilos de carácter: Afectan a palabras sueltas (énfasis, código, cita).
  • Estilos de página: Definen márgenes, encabezados y pies de página.

📄 Plantillas

  • Una plantilla es un documento que contiene estilos predefinidos.
  • Al crear un documento desde una plantilla, hereda los estilos.
  • Ejemplos: carta comercial, informe académico, currículum.

⏱️ 110 min
RA3 CE-b · Crear, modificar y manipular documentos (15%)
🎥 Vídeo · Índices automáticos
📌 Encabezado y pie de página

Encabezado: Zona superior de la página. Suele contener el título del documento o el nombre de la empresa.
Pie de página: Zona inferior. Contiene número de página, fecha o autor.
Inserción: Insertar → Encabezado y pie de página → Elegir estilo.

🔑 Concepto clave

Un índice automático (Tabla de contenidos) se genera a partir de los estilos de título (Título 1, Título 2...). Cuando modificas el documento, puedes actualizar el índice con un clic. No es necesario escribirlo manualmente.

📑 Insertar índice automático

  • Posiciona el cursor donde quieras el índice
  • Insertar → Tabla de contenido y índice → Tabla de contenido
  • Elige el nivel de profundidad (hasta Título 3, etc.)

🔄 Actualizar índice

  • Botón derecho sobre el índice → Actualizar índice/Actualizar todo
  • También desde el menú contextual de la tabla

🎥 Vídeo · Qué es una hoja de cálculo
📌 Conceptos fundamentales de Calc
  • Libro de cálculo: Archivo .ods que contiene varias hojas.
  • Hoja: Cada pestaña dentro del libro (Hoja1, Hoja2...).

  • .xlsx: Formato de Microsoft Excel
  • .csv: Valores separados por comas

⌨️ Navegación básica

  • Flechas: Moverse entre celdas
  • Tab: Avanza una columna a la derecha
  • Enter: Avanza una fila hacia abajo
  • Ctrl + Inicio: Ir a A1

B50

Calc · Fórmulas aritméticas · Funciones SUMA, PROMEDIO, CONTAR

⏱️ 110 min
RA3 CE-e · Operaciones básicas con hoja de cálculo (35%)
🎥 Vídeo · Fórmulas y funciones
📌 Fórmulas y funciones

Fórmula: Operación que introducimos en una celda, empezando por =. Ej: =A1+B1
Función: Fórmula predefinida. Ej: =SUMA(A1:A10). Las funciones se escriben en MAYÚSCULAS.

➕ Operadores aritméticos

  • + Suma
  • - Resta
  • * Multiplicación
  • / División
  • ^ Potencia

📊 Funciones básicas

  • =SUMA(rango) → Suma valores
  • =PROMEDIO(rango) → Media aritmética
  • =CONTAR(rango) → Cuenta números
  • =MAX(rango) → Valor máximo
  • =MIN(rango) → Valor mínimo
🔑 Concepto clave

Al copiar una fórmula a otras celdas, Calc ajusta automáticamente las referencias relativas (A1 se convierte en A2, A3...). Si quieres que una celda NO cambie, usa referencias absolutas con el signo $ (ej: $A$1).

B51

Calc · Función condicional SI · Toma de decisiones en hojas de cálculo

⏱️ 110 min
RA3 CE-e · Operaciones básicas con hoja de cálculo (35%)
🎥 Vídeo · Función SI
📌 Sintaxis de la función SI

=SI(condición; valor_si_verdadero; valor_si_falso)

Ejemplo: =SI(A1>=5; "Aprobado"; "Suspenso")
Si A1 es mayor o igual a 5, escribe "Aprobado"; si no, escribe "Suspenso".

🔄 Operadores de comparación

  • = Igual
  • <> Distinto
  • > Mayor que
  • < Menor que
  • >= Mayor o igual
  • <= Menor o igual

🧠 SI anidado

Puedes poner un SI dentro de otro: =SI(A1>=9; "Sobresaliente"; SI(A1>=7; "Notable"; "Aprobado"))

Introducción a LibreOffice Base · Bases de datos, tablas y registros

⏱️ 110 min
RA3 CE-e · Operaciones básicas con hoja de cálculo y base de datos (35%)
🎥 Vídeo · Qué es una base de datos
📌 Conceptos de bases de datos
  • Base de datos: Conjunto de datos organizados para su fácil acceso y gestión.
  • Tabla: Conjunto de datos sobre un tema específico (ej: tabla Clientes).
  • Registro (fila): Cada entrada individual en la tabla (un cliente).
  • Campo (columna): Cada atributo del registro (nombre, teléfono, email).
  • Clave principal (PK): Campo que identifica cada registro de forma única (ej: ID_cliente).

📁 Tipos de campo comunes

  • TEXTO/VARCHAR: Nombres, direcciones
  • NÚMERO/INTEGER: Edades, cantidades
  • FECHA/DATE: Fechas de nacimiento, pedidos
  • MONEDA/CURRENCY: Precios, salarios

📂 Extensiones de Base

  • .odb: Archivo de base de datos de LibreOffice
  • Puede trabajar con motores: HSQLDB (embebido) o MySQL, PostgreSQL

📌 ¿Qué es una consulta?

Una consulta es una pregunta que hacemos a la base de datos. Devuelve un subconjunto de registros que cumplen ciertas condiciones. Las consultas se guardan y se pueden reutilizar.

🔍 Crear una consulta (diseño)

  2. Selecciona la tabla que quieras consultar
  3. Arrastra los campos que quieras mostrar
  4. Define criterios (Filtros)
  5. Ordena por algún campo (Orden ascendente/descendente)
  6. Ejecuta y guarda

📌 ¿Qué es un formulario?

Un formulario es una interfaz gráfica que facilita la introducción, modificación y visualización de datos en una tabla. Es más amigable que trabajar directamente en la tabla. Se usa para que los usuarios no técnicos puedan manejar la base de datos.

  4. Selecciona diseño (columnas, tabla)
  5. Establece formato de fondo y colores

✏️ Tipos de campos en formularios

  • Formatted Field: Para números con formato
  • Date Field: Para fechas (calendario)
  • List Box: Para elegir de una lista
  • Check Box: Para Sí/No

Base · Relaciones entre tablas · Claves foráneas · Integridad referencial

⏱️ 110 min
RA3 CE-e · Operaciones básicas con base de datos (35%)
🎥 Vídeo · Relaciones en Base
📌 Tipos de relaciones
  • 1 a N (uno a muchos): Un registro de la tabla A puede tener muchos registros en la tabla B (ej: 1 cliente → muchos pedidos).
  • 1 a 1 (uno a uno): Un registro en A se relaciona con solo uno en B.
  • N a N (muchos a muchos): Requiere una tabla intermedia.

🔗 Crear una relación

  1. Herramientas → Relaciones
  2. Añade las tablas que quieras relacionar
  3. Arrastra el campo clave de una tabla a la otra
  4. Elige tipo de relación (1 a N por defecto)
  5. Activa "Integridad referencial" para evitar datos huérfanos

🔐 Integridad referencial

Evita que se borre un cliente si tiene pedidos asociados. También evita crear pedidos de un cliente que no existe.

Impress · Transiciones entre diapositivas y animaciones de objetos

⏱️ 110 min
RA3 CE-g · Elaborar presentaciones multimedia con normas de diseño (20%)
🎥 Vídeo · Transiciones y animaciones
📌 Transición entre diapositivas

Transición: Efecto visual al pasar de una diapositiva a la siguiente.
Diapositivas → Transición de diapositiva. Puedes elegir disolvencia, cortinilla, empujar, etc. Configura duración y si es automática o por clic.

✨ Animaciones de objetos

Se aplican a elementos individuales (texto, imágenes). En vista Normal → Animación Personalizada. Puedes elegir entrada, énfasis, salida y trayectorias.

Ejemplo: Un título puede "entrar volando", un párrafo "aparecer con disolvencia".

🎬 Consejos de diseño

  • No abuses de las animaciones (pueden distraer).
  • Usa transiciones suaves (disolvencia, empujar).
  • Mantén la coherencia: mismo tipo de transición en toda la presentación.
  • Las animaciones deben ayudar a contar la historia, no al revés.
